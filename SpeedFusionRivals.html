Speed Fusion Rivals

About

Speed Fusion Rivals is a fast-paced racing game developed in Unreal Engine using Blueprints. The game features intense racing action with a focus on speed, skill, and strategy.

Project Info

  • 👤 Role: Gameplay Programmer
  • 👥 Team Size: 1
  • 🕒 Time frame: 3 months
  • 🛠 Engine: Unreal Engine (Blueprints)

Introduction

Speed Fusion Rivals was the first project I worked on within my first year of university. it was a solo project I developed in Unreal Engine using Blueprints. The game is a fast-paced racing experience where the player competes in intense races, with a focus on speed, skill, and strategy. The focus of this project was to learn the basics of Unreal Engine and Blueprints, as well as to create a small but polished game experience.

What I Had Learned

Through the development of Speed Fusion Rivals, I learned the basics of Unreal Engine and Blueprints, as well as how to create a small but polished game experience. I also learned how to create a fast-paced racing experience and how to use various gameplay mechanics to create an engaging and challenging game. Overall, this project was a great learning experience for me and helped me to develop my skills in game development.
