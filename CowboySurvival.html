Cowboy Survival

A top-down wave survival game focused on movement, shooting, enemy behaviour, and game-state systems.

About

Cowboy Survival was a solo first-year university project that helped me apply C++ programming concepts inside Unreal Engine.

Project Info

1

Introduction

The project was a top-down survival game where the player had to survive against waves of enemies. The player could move around the map and shoot enemies to stay alive. It was a useful project for turning classroom programming concepts into a playable Unreal Engine prototype.

What I Learned

I learned the fundamentals of C++ programming in a game development context. I gained experience creating gameplay mechanics, implementing AI behaviour, managing game states, debugging code, and improving systems inside Unreal Engine.
