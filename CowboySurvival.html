Cowboy Survival

About

this was a solo project I worked on in my first year of university as part of learning C++ programming and Unreal Engine.

Project Info

  • 👤 Role: Gameplay Programmer
  • 👥 Team Size: 1
  • 🕒 Time frame: 3 months
  • 🛠 Engine: Unreal Engine (C++)

Introduction

The project was a top down survival game where the player had to survive against waves of enemies. The player could move around the map and shoot at the enemies to survive. The game was developed using Unreal Engine and C++ programming language. The project was a great learning experience for me as it allowed me to apply the programming concepts I had learned in class to a real project. I also learned how to use Unreal Engine and its various features to create a game. Overall, it was a great project that helped me develop my skills as a game developer.

What I Had Learned

Through this project, I learned the fundamentals of C++ programming within the context of game development. I gained experience in creating gameplay mechanics, implementing AI behavior, and managing game states. Additionally, I improved my problem-solving skills and learned how to debug and optimize code in Unreal Engine.
